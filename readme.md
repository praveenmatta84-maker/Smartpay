# 💳 SmartPay – Enterprise Digital Banking System

SmartPay is a full-stack digital banking application developed using Spring Boot and React. It provides secure banking operations such as wallet management, money transfers, transaction history, and role-based administration through a modern web interface.

---

## 🚀 Features

### 👤 User Module
- User Registration
- Secure Login (JWT Authentication)
- Profile Management
- Wallet Balance
- Add Money
- Transfer Money
- Transaction History

### 👨‍💼 Admin Module
- Admin Login
- Dashboard Statistics
- View All Users
- View Transactions
- Delete Users
- Monitor Total Credits & Debits

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/smartpay.git
```

### Backend

```bash
cd backend/backend
./mvnw spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
